# DRF-To-Do-List

A simple to-do list made to practice Django REST Framework.

Functionality: Add a new Task; Edit a Task; Delete a Task.

Frontend: CSS, Bootstrap 5.1.

Backend: Django 4.0, Django REST Framework, AJAX.
